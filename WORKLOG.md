# WORKLOG — 작업 기록 (chatlog)

이 앱을 만들면서 **무엇을, 왜, 어떻게** 바꿨는지 시간순으로 남기는 곳.
새 작업을 끝낼 때마다 맨 위에 한 덩어리 추가한다. (최신이 위)

형식:
```
## YYYY-MM-DD  제목
- 한 일:
- 왜:
- 어떻게 확인:
- 다음 할 일:
```
